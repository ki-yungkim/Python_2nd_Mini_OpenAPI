- 5. 상품 가격 최고, 최소, 평균 출력
- 6. 상품 원플러스원 판매점 검색

- 9. 목록에서 검색 기능
- 10. 할인 빈도 (할인 Y 갯수로)  
- 11. 원플러스원 빈도 (1+1 Y 갯수로)
